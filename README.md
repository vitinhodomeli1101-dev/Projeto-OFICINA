Contexto

Este projeto consiste no desenvolvimento de uma interface web para uma oficina mecânica, com o objetivo de apresentar serviços, 
facilitar o contato com clientes e oferecer uma experiência digital moderna e acessível. 
A aplicação será construída utilizando HTML5 e CSS3, seguindo boas práticas de semântica, organização e acessibilidade.

Publico Alvo 

O sistema é destinado a clientes que possuem veículos e necessitam de serviços automotivos.
O público principal inclui adultos entre 25 e 60 anos, que utilizam dispositivos móveis para 
localizar serviços mecânicos. 

Dor do usuário 

Atualmente, os clientes enfrentam dificuldades para encontrar informações claras 
sobre serviços, horários de funcionamento e formas de contato da oficina.

Critério de sucesso

O usuário consegue acessar o site, localizar rapidamente
os serviços oferecidos e entrar em contato com a oficina
de forma simples e intuitiva em menos de 2 minutos.
